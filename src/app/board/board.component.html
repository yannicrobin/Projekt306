Resultatdarstellung

Am Schluss soll ein Prototyp vorhanden
sein, der zeigt, wie das Endprodukt aussehen wird. Es wird eine ähnliche Darstellung wie das Endprodukt haben und die Funktionalitäten, des Endprodukts werden vorhanden sein.

Der Entwickler sollte ein Mockups anfertigen, damit jeder weiss, wie das Endprodukt aussehen sollte.
Als Entwickler möchte ich zuerst einen Prototyp erstellen, um mögliche Probleme frühzeitig zu erkennen.
Als Entwickler möchte ich die Webseite programmieren, so dass sie den Anforderungen entspricht.

Der Prototyp muss bis Ende 2019 designt, fertiggestellt, und getestet sein.

Anhand eines Mockups, dass mit dem Programm „Balsamiq Mockups“ erstellt wird, kann man sehen, wie der Prototyp, schlussendlich aussehen wird. Der Prototyp wird eine Webseite sein, die das Endprodukt darstellen soll.
